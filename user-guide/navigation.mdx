Twenty's interface is designed for speed and efficiency. Master navigation to work faster and get more done.

## Main Navigation

The left sidebar provides access to all major sections of Twenty:

  <Card title="Dashboard" icon="house">
    Your workspace overview with quick stats and recent activity
  </Card>
  
  <Card title="Objects" icon="database">
    Access all your data objects (Companies, People, Opportunities, etc.)
  </Card>
  
  <Card title="Favorites" icon="star">
    Quick access to frequently used views and records
  </Card>
  
  <Card title="Settings" icon="gear">
    Workspace configuration, data model, and integrations
  </Card>
</CardGroup>

### Navigation Menu

The sidebar shows all available objects in your workspace:

- **Standard objects** - Companies, People, Opportunities (built-in)
- **Custom objects** - Objects you've created for your business
- **Views** - Saved views appear under each object

<Tip>
Click the object icon to expand and see all views for that object.
</Tip>

### Favorites

Pin important views and records for quick access:

1. Navigate to any view or record
2. Click the star icon in the header
3. Access from the **Favorites** section in the sidebar

To remove from favorites, click the star icon again.

## Search

The global search helps you find records quickly across all objects.

### Using Search

<Steps>
  <Step title="Open search">
    Press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux) to open the search interface.
  </Step>
  
  <Step title="Enter search terms">
    Type any part of a record name, email, company, or other searchable field.
  </Step>
  
  <Step title="Select a result">
    Use arrow keys to navigate results and press `Enter` to open a record.
  </Step>
</Steps>

### Search Tips

- Search looks across all objects by default
- Results update as you type
- Recent records appear first when search is empty
- Press `Esc` to close search

## Command Menu

The command menu provides quick access to actions and navigation.

### Opening the Command Menu

Press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux) - the same shortcut as search. The menu adapts to your current context.

### Available Commands

<Tabs>
  <Tab title="Navigation">
    - Go to any object
    - Switch between views
    - Access settings sections
    - Open recent records
  </Tab>
  
  <Tab title="Actions">
    - Create new records
    - Execute workflows
    - Export data
    - Manage selections
  </Tab>
  
  <Tab title="Search">
    - Find records
    - Filter results by object
    - Jump to specific fields
  </Tab>
</Tabs>

### Command Menu Navigation

- Type to filter commands
- Use arrow keys to navigate
- Press `Enter` to execute
- Press `Esc` to close

<Info>
The command menu shows different options based on your current location and selected records.
</Info>

## Keyboard Shortcuts

Work faster with keyboard shortcuts.

### Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+K` | Open command menu/search |
| `Cmd/Ctrl+N` | Create new record |
| `G` then `H` | Go to home/dashboard |
| `G` then `S` | Go to settings |
| `?` | Show keyboard shortcuts |

### Table View Shortcuts

| Shortcut | Action |
|----------|--------|
| `↑` `↓` | Navigate between rows |
| `←` `→` | Navigate between columns |
| `Enter` | Edit selected cell |
| `Esc` | Exit edit mode |
| `Cmd/Ctrl+A` | Select all rows |
| `Space` | Toggle row selection |

### Kanban View Shortcuts

| Shortcut | Action |
|----------|--------|
| `↑` `↓` | Navigate between cards |
| `←` `→` | Navigate between columns |
| `Enter` | Open selected card |
| `Cmd/Ctrl+D` | Duplicate card |

<Tip>
Press `?` anywhere in Twenty to see all available keyboard shortcuts for your current context.
</Tip>

## Record Navigation

When viewing a record:

### Record Header

- **Record name** - Editable at the top
- **Actions menu** - Access record operations
- **Favorite star** - Add to favorites
- **More options** - Delete, duplicate, share

### Record Tabs

Switch between different aspects of a record:

- **Summary** - Key information and fields
- **Timeline** - Activity history and notes
- **Relations** - Connected records
- **Files** - Attached documents

### Related Records

The right sidebar shows related records:

- Quickly view connections
- Click to navigate to related records
- Add new relations inline

## Quick Actions Bar

The bottom action bar appears when you select records:

- **Bulk edit** - Update multiple records at once
- **Delete** - Remove selected records
- **Export** - Download selected records
- **Assign** - Change record ownership
- **Run workflow** - Execute workflows on selection

## Customizing Your Navigation

### Reorder Navigation Items

1. Go to **Settings → Objects**
2. Drag objects to reorder them
3. Changes reflect immediately in the sidebar

### Hide Unused Objects

1. Go to **Settings → Objects**
2. Click on an object
3. Toggle **Show in navigation** to hide it

<Warning>
Hiding an object from navigation doesn't delete it or restrict access. Users can still access it via search or direct links.
</Warning>

## Mobile Navigation

On mobile devices:

- Tap the menu icon (☰) to open the sidebar
- Swipe right to access navigation
- Pull down to refresh data
- Tap and hold for quick actions
